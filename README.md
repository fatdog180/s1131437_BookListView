# 圖書管理系統 (BookListView)

一個基於 C# Windows Forms 開發的智慧圖書管理與模擬借閱系統。本專案透過 `ListView` 控制項展示書籍資訊，結合多種圖示檢視模式切換與事件驅動處理，提供直覺且流暢的使用者操作體驗。

## 📌 功能特點

1. **多功能清單檢視 (`ListView`)**
   - 結合雙 `ImageList`（大圖示 `90x120`、小圖示 `15x20`），動態綁定 8 本經典文學書籍封面。
   - 支援完整的資料行欄位配置，包含「書名」、「作者」與「類別」。

2. **動態檢視模式切換 (`ComboBox`)**
   - 提供 5 種標準檢視模式即時切換：大圖示 (`LargeIcon`)、詳細資料 (`Details`)、小圖示 (`SmallIcon`)、清單 (`List`)、大圖示加詳細資料 (`Tile`)。

3. **智慧借閱與防呆機制**
   - 雙擊書籍項目可觸發 `ItemActivate` 事件，彈出互動式 `MessageBox` 確認方塊。
   - 整合防呆邏輯，若書籍已存在於右側「借書清單」(`ListBox`) 中，將會自動攔截，避免重複借閱。

4. **介面呈現優化**
   - 資料載入時採用 `BeginUpdate()` 與 `EndUpdate()` 方法，暫停控制項重繪，完美解決大量資料匯入時的畫面閃爍問題。

## 🛠️ 開發環境

- **程式語言**：C#
- **開發平台**：Windows Forms (WinForm)
- **開發工具**：Visual Studio 2022
- **目標框架**：.NET Framework / .NET Core

## 🚀 使用說明

1. **啟動程式**：執行專案後，系統將自動載入 8 本書籍，預設以「大圖示」模式排列。
2. **切換檢視**：點擊右上角的「檢視方式」下拉式選單，選擇不同的項目（例如切換至「詳細資料」可查看作者與類別欄位）。
3. **模擬借閱**：在左側的書籍清單中「雙擊」任何一本書，畫面會跳出確認視窗。點選「是」之後，該書籍就會被加到右側的「借書清單」中。
